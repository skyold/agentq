### 触发逻辑

```python
# 两种触发方式：

# 1. 定时触发（fallback）
if time_diff >= self.trigger_interval:
    self.last_trigger_at = event_time
    self.running = True
    return True

# 2. 信号触发（优先级更高）
if signal_pool_triggered and not self.running:
    self.last_trigger_at = event_time
    self.running = True
    return True
```

### 后端处理

```python
@app.websocket("/ws")
async def websocket_endpoint(websocket: WebSocket):
    await websocket.accept()
    
    while True:
        data = await websocket.receive_json()
        msg_type = data.get("type")
        
        if msg_type == "bootstrap":
            # 初始化用户和账户
            await handle_bootstrap(websocket, data)
        
        elif msg_type == "get_snapshot":
            # 发送当前数据快照
            await send_snapshot(websocket, data)
        
        elif msg_type == "place_order":
            # 下单
            await handle_order(websocket, data)
```

## 总结

Hyper Alpha Arena 的核心模块设计遵循以下原则：

1. **模块化**: 每个功能模块独立，通过清晰的接口通信
2. **可扩展**: 插件化设计（因子、技能、工具）
3. **实时性**: WebSocket 双向通信，毫秒级数据更新

- 快速添加新因子和策略
