# Hyper Alpha Arena 数据库设计

本文档详细解析项目的数据库架构、表结构和数据关系。

## 数据库架构

### 双数据库设计

项目使用两个 PostgreSQL 数据库：

1. **主数据库** (`alpha_arena`): 存储业务数据
2. **快照数据库** (`alpha_snapshots`): 存储资产快照数据（用于资产曲线计算）

```python
# backend/database/connection.py

DATABASE_URL = os.getenv("DATABASE_URL")
SNAPSHOT_DATABASE_URL = os.getenv("SNAPSHOT_DATABASE_URL")

engine = create_engine(DATABASE_URL)
snapshot_engine = create_engine(SNAPSHOT_DATABASE_URL)
```

### 设计原则

1. **数据隔离**: 快照数据独立存储，避免影响主库性能
2. **软删除**: 关键表使用 `is_deleted` 标记，支持数据恢复
3. **环境隔离**: Testnet/Mainnet 数据通过 `environment` 字段隔离
4. **审计追踪**: 所有关键操作记录日志（AI 决策、交易执行）
5. **版本管理**: 通过 migration 系统管理 schema 变更

### 2. 账户和交易配置

#### `accounts` - 交易账户表

```python
class Account(Base):
    __tablename__ = "accounts"
    
    id: int  # PK
    user_id: int  # FK -> users.id
    version: str  # "v1"
    
    # 账户身份
    name: str  # 显示名称（如 "GPT Trader"）
    account_type: str  # "AI" | "MANUAL"
    is_active: str  # "true" | "false"
    auto_trading_enabled: str  # "true" | "false"
    
    # AI 模型配置
    model: str  # AI 模型名称
    base_url: str  # API 端点
    api_key: str  # API Key（加密存储）
    
    # 账户余额
    initial_capital: Decimal  # 初始资金
    current_cash: Decimal  # 当前现金
    frozen_cash: Decimal  # 冻结资金
    
    # Hyperliquid 配置
    hyperliquid_enabled: str  # "true" | "false"
    hyperliquid_environment: str  # "testnet" | "mainnet" | null
    hyperliquid_testnet_private_key: str  # 加密存储
    hyperliquid_mainnet_private_key: str  # 加密存储
    max_leverage: int  # 最大杠杆（1-50）
    default_leverage: int  # 默认杠杆
    
    # 仪表盘可见性
    show_on_dashboard: bool
    
    # 软删除
    is_deleted: bool
    deleted_at: datetime
    
    created_at: datetime
    updated_at: datetime
    
    # Relationships
    user: User
    positions: List[Position]
    orders: List[Order]
    prompt_binding: AccountPromptBinding
```

**说明**:
- 支持多账户（每个 AI Trader 一个账户）
- 私钥和 API Key 使用加密存储
- `hyperliquid_environment` 决定连接哪个网络

## 数据库索引优化

### 主要索引

```sql
-- 时间范围查询优化
CREATE INDEX idx_crypto_klines_timestamp ON crypto_klines(timestamp);
```

### 加密实现

```python
# backend/utils/encryption.py

from cryptography.fernet import Fernet
import bcrypt

def encrypt_private_key(value: str) -> str:
    """使用 Fernet 对称加密"""
    key = get_encryption_key()
    f = Fernet(key)
    return f.encrypt(value.encode()).decode()

def decrypt_private_key(encrypted: str) -> str:
    """解密私钥"""
    key = get_encryption_key()
    f = Fernet(key)
    return f.decrypt(encrypted.encode()).decode()

def hash_password(password: str) -> str:
    """bcrypt 哈希密码"""
    return bcrypt.hashpw(password.encode(), bcrypt.gensalt()).decode()

def verify_password(password: str, hashed: str) -> bool:
    """验证密码"""
    return bcrypt.checkpw(password.encode(), hashed.encode())
```
